Dr. Visalam Vasanth PT, DPT!

My dream would be to work in user experience and front end web development. My long term goal is to combine my passion for web development with my passion for pediatric physical therapy by implementing more efficient web based applications to help resolve some of the challenges faced in pediatric healthcare today.
